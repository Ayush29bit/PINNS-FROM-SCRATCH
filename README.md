# Physics-Informed Neural Networks (PINNs) From Scratch
Research-oriented implementation of Physics-Informed Neural Networks (PINNs) in PyTorch for scientific machine learning and PDE solving.

## Overview

This repository contains research-oriented implementations of Physics-Informed Neural Networks (PINNs) using PyTorch for solving ordinary differential equations (ODEs) and partial differential equations (PDEs).

The project explores how neural networks can incorporate physical laws directly into the optimization objective using automatic differentiation and physics-constrained loss functions.
